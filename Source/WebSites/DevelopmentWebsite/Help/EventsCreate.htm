Only the Tribute creators and administrators can create an event.

To create an event:

1. Go to the events page by clicking the "Events" tab.
2. Log in to be able to create an event. (Note that you can only create events for tributes you have created or are the administrator of)
3. Select "Add an Event" from the "Tribute Tools".
4. Enter all of the required event information and click "Save and Invite Guests" to invite guests to your event.
