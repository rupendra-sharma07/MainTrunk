Only the Tribute creators and administrators can edit an event.

To edit an event:

1. You must be viewing the event would like to edit.
2. Log in to be able to edit the event.
3. Select "Edit Event" from the "Tribute Tools"
4. Edit the event details.
5. Click the "Save Event" button located at the bottom of the page.
6. A dialog box will appear confirming the changes to your event.
