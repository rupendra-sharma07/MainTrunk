Members can only delete guestbook messages they add. Tribute creators and administrators can delete all guestbook messages added to their tribute.

To delete a guestbook message:

1. You must be viewing the guestbook message you would like to remove.
2. Log in to be able to delete the guestbook message.
3. Click the "Delete" button next to the guestbook message.
4. A dialog box will appear confirming the deletion of your guestbook message.
