To edit an album:
1. You must be viewing the photo album you would like to edit.
2. Log in to be able to edit the album.
3. Select "Edit Album" from the "Tribute Tools".
4. Edit the "Album Name" and/or "Album Description".
5. Click the "Save Album" button located at the bottom of the page.
6. A dialog box will appear confirming the changes to your album.



To edit a photo:
1. You must be viewing the photo you would like to edit.
2. Log in to be able to edit the photo.
3. Select "Edit this photo" from the "Tribute Tools".
4. Edit the "Photo Name" and/or "Photo Description".
5. Click the "Save Photo" button located at the bottom of the page.
6. A dialog box will appear confirming the changes to your photo.
